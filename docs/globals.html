strToNum: Decoder<number, unknown> = chain(string,predicate(/^\d+(\.(\d*))?$/.test, 'expected number', ExpectedNumber),map(parseFloat),)

Const text

text: TextDecoder = Object.assign(textOptions(), {options: textOptions,})
  • Defined in src/composite/text.ts:64

A decoder which can decode a string, with additional options for validation. The default decoder provides sane defaults for decoding text: trim whitespace from start and end, minimum length of 1 and no maximum.

example
// text decoder with default options
const decoder = text;

// specify a maximum length (default unlimited)
const decoderMax = text.options({ maxLength: 10 });

// don't trim whitespace from start and end (default true)
const decoderNoTrim = text.options({ trim: false });

// no minimum length (default 1)
const decoderNoMin = text.options({ minLength: 0 })

choose

  • choose<T>(...options: T): CombinedTypeOf<T>
    • Defined in src/composite/choose.ts:41

    Create a decoder which can succesfully decode an input value using one of the supplie decoders. The decoders are invoked in the given order until one is successful. If none are successful, the errors from all decoders are concatenated and returned.

    example
    const choice = choose(number, string); // is a Decoder<number|string>;

const result1 = choice(1); // = { ok: true, value: 1 }
const result2 = choice('hello'); // = { ok: true, value: 'hello' }

// the `error` field will contain the errors of all decoders
const result3 = choice(false); // = { ok: false, error: [ ... ] }

    Type parameters

    Parameters

      The possible decoders to try.

maybe

  • maybe<Out, In>(decoder: Decoder<Out, In>): Decoder<Out | undefined, In>
    • Defined in src/composite/maybe.ts:30

    Wraps another decoder to permit an empty string, null, or undefined in addition to the types decoded by the wrapped decoder. If the input value or the decoded value from the wrapped decoder is an empty string, null, or undefined, then the decoder will return a value of undefined.

    example
    const decoder = maybe(string);

const result1 = decoder(''); // will be { ok: true, value: undefined }
const result2 = decoder(null); // will be { ok: true, value: undefined }
const result3 = decoder(undefined); // will be { ok: true, value: undefined }
const result4 = decoder('hello'); // will be { ok: true, value: 'hello' }
const result5 = decoder(42); // will be { ok: false, error: [...] }

    Type parameters

    • Out

      The output value type of the wrapped decoder

    • In

      The input value type of the wrapped decoder

    Parameters

    • decoder: Decoder<Out, In>

      The decoder which will decode non-empty values

    Returns Decoder<Out | undefined, In>

    A new decoder which can decode an empty string, null or undefined, as well as anything that the wrapped decoder can decode.
